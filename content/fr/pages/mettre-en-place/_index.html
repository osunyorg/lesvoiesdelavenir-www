Comment mettre en place les voies de l'avenir dans votre ville ?

body: >-

Le format

Vous pouvez mettre en place Les voies de l'avenir physiquement au travers de panneaux informationnels. Pour chaque femme, un panneau a été réalisé avec un QR code redirigeant vers sa page du site internet.De plus, un livret vous permet de proposer une expérience ludique à destination des lycéens. Il contient un jeu de piste entre les différentes femmes, modifiable pour l'adapter à votre ville.

Les affiches
Les affiches L'affiche est au format A3. Une affiche présente une femme et mentionne dans un premier temps son prénom, son nom, sa date de naissance et de décès et le domaine du métier exercé par la femme. Un texte explicatif est rédigé, il contient un résumé du texte présent sur le site, accessible via le QR code ou le lien sur l'affiche.
Plusieurs formats d'impressions sont possibles. Pour l'expérience pilote sur Pauillac, nous avons opté pour un format semi-permanent, pour pouvoir ainsi retirer le contenu ou le changer. Cela vous permet d'organiser l'expérience de façon évènementiel, si vous souhaitez récupérer l'espace des affiches pour un autre projet plus tard.
Cependant, le format est aussi réalisable de manière permanente.

Le livret Ce livret est un template qui permet de créer un parcours interactif et ludique entre les affiches et le site. Il s'organise comme un jeu de piste, où chaque page vous propose de résoudre des énigmes pour progresser.Vous trouverez également un PDF contenant un guide et des conseils pour créer les énigmes.
